# What is a Program?

A program is a set of instructions written for a computer to perform a specific task.

- Instructions are written in a programming language.
- A computer follows these instructions step by step to produce output.

### Examples
- Calculator Program
- Login System Program
- Browser Program

---

# What is a Programmer?

A programmer is a person who creates computer programs.

- Writes instructions using programming languages.
- Tests programs.
- Fixes errors in programs.

### Main Work
- Writing Code
- Debugging Errors
- Building Software and Applications

---

# What is a Programming Language?

A programming language is a formal language used to communicate with a computer.

- Allows humans to write instructions.
- Instructions are converted into machine-readable form.

### Examples
- Python
- JavaScript
- Java
- C++

---

# Types of Programming Languages

Programming languages are divided into:

1. Low Level Language
2. High Level Language

---

# Low Level Language

Low level languages are close to hardware.

- Difficult for humans to understand.
- Fast for computers to execute.

### Examples
- Machine Language
- Assembly Language

---

# High Level Language

High level languages are easy to read and write.

- Close to human language.
- Easier for programmers to learn and use.

### Examples
- Python
- JavaScript
- Java
- C++

---

# Types of Low Level Languages

1. Machine Language
2. Assembly Language

---

# Machine Language

Machine language uses binary code (0s and 1s).

- Directly understood by the computer.

### Characteristics
- Fast Execution
- Hard to Understand
- No Translator Needed

### Example

```text
10101000 01000001
```

---

# Assembly Language

Assembly language uses symbolic instructions.

- Requires an assembler to convert into machine code.

### Example

```assembly
MOV A, 5
ADD A, 2
```

### Characteristics
- Easier than Machine Language
- Uses Mnemonics
- Used in System Programming
